# Quad Dodge

Simple 2D game made with OpenGL.

![Screenshot](Screenshots/example.png)

## Libraries Used

- GLFW (for window and input)  
- GLEW (to use OpenGL functions)  
- GLM (for maths)  

## How to Build
```bash
git clone https://github.com/VoidSoup42/Quad_Dodge.git
cd Quad_Dodge
mkdir build
cd build
cmake ..
make
./Quad_Dodge
